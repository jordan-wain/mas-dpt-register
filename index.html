How This Report Is Generated

1. Query MAS Financial Institutions Directory API — filter to Major Payment Institutions with DPT activity

2. Paginate through all results to get the full licensed entity list

3. Compare licensed list against exempt entity seed data to detect graduations

4. Diff against previous snapshot for new/removed licensees

5. Generate this HTML and push to GitHub Pages

Note: Exempt entity data is manually seeded from MAS website screenshots (the MAS website is a client-rendered SPA that cannot be scraped). Licensed data is scraped live from the MAS FID API on every run.
